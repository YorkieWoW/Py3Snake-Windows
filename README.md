# Py3Snake-Windows
Ported Py3Snake to run on Windows with no external dependencies.

Original Linux version can be found on my Github under Py3Snake

<img width="627" height="613" alt="py3_snake_windows" src="https://github.com/user-attachments/assets/5d0e79ff-1f99-4ce1-8e0b-4ef923be342f" />

Features retro dark & neon green colour pallete, score counter and more.

To run it, simply double click snake_windows.py or navigate to download location and run "python snake_windows.py". Enjoy!
